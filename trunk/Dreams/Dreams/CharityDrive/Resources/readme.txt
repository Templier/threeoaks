Dream for the Wincustomize Charity Drive
(C) 2007, Julien Templier
----------------------------------------

This is a dynamic dream build specially for the Wincustomize Charity Drive. 

It randomly rotates through included photos using slow zooming and panning effects, and fading transitions.

If you like this dream, consider supporting Wincustomize by participing to the Charity Drive (http://www.wincustomize.com/charity.aspx).
