DockletSetLabel

Changes the current label of the specified Docklet.

int DockletSetLabel(
   HWND hwndDocklet,
   char *szLabel
);

Parameters

hwndDocklet
The handle to the window passed during OnCreate that owns this docklet and therefore identifies the docklet to the host.
szLabel
[in] Pointer to a string with a new label for the specified docklet.

Return Value

Returns the length of the string that either was or would be copied to szLabel including the null character.
