OnConfigure

Called by the host in the event OnRightButtonClick returned FALSE or was not implemented, and it was selected to configure the dock item. If you do not implement this function or return FALSE, ObjectDock will, as a default handler, will open a basic configuration dialog allowing the dock item's text and image to be changed.
To override this functionality, implement it and return TRUE.

BOOL CALLBACK OnConfigure(
   DOCKLET_DATA *lpData
);
lpData
Personal data for this plugin instance returned during OnCreate. Datatype is arbitrary.

Return Value

Return TRUE to signify that this event was acted upon.
Return FALSE or do not implement to signify that this event was NOT acted upon.
In this case, for OnConfigure, a default configuration dialog will be shown, which will allow the user to change the docklet's caption and image.
