OnAcceptDropFiles

Gives the opportunity to a docklet to express its ability to accept dropped files. Implementing this function and returning TRUE will allow for the possibility for files to be dropped onto this dock item, and thus a call to OnDropFiles.

BOOL CALLBACK OnAcceptDropFiles(
   DOCKLET_DATA *lpData
);
lpData
Personal data for this plugin instance returned during OnCreate. Datatype is arbitrary.

Return Value

Return TRUE to signify that this docklet can accept files dropped onto it
Return FALSE or do not implement to signify that this docklet does not accept dropped files. (Default)
