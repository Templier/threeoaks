OnLeftButtonClick

Called by the host when the dock item that represents this docklet is clicked. If you choose to implement this function, you should return True to indicate you processed the message.

BOOL CALLBACK OnLeftButtonClick(
   DOCKLET_DATA *lpData,
   POINT *ptCursor,
   SIZE *sizeDocklet
);

Parameters

lpData
Personal data for this plugin instance returned during OnCreate. Datatype is arbitrary.
ptCursor
The position of the cursor at the time of the event, in coordinates relative to the top-left of the docklet image.
sizeDocklet
The size of the docklet image at the time of the event.

Return Value

Return TRUE to signify that this event was acted upon.
Return FALSE or do not implement to signify that this event was NOT acted upon.
In this case, for OnLeftButtonClick, no default action will be taken
