ObjectDock SDK Reference

OnDestroy

Sent right before a plugin is finished being destroyed. The plugin should take this opportunity to do cleaning up operations it requires.

void CALLBACK OnDestroy(
   DOCKLET_DATA *lpData
   HWND hwndDocklet
);
lpData
Personal data for this plugin instance returned during OnCreate. Datatype is arbitrary.
hwndDocklet
The handle to the window passed during OnCreate that owns this docklet and therefore identifies the docklet to the host.
