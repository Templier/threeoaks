DockletLockMouseEffect

Allows a Docklet to lock the mouse effect (e.g. zooming, etc) of its owning dock, something useful when trying to display a right-click menu or otherwise from a docklet.

void DockletLockMouseEffect(
   HWND hwndDocklet,
   BOOL bLock
);

Parameters

hwndDocklet
The handle to the window passed during OnCreate that owns this docklet and therefore identifies the docklet to the host.
bLock
TRUE to lock the mouse-over effect (zooming, etc) of the dock
FALSE to unlock the mouse-over effect (zooming, etc) of the dock

Remarks

The mouse-over effect lock is done via a counter, so that if multiple Docklets are locking at the same time the effect will stay locked until the last Docklet unlocks. The counter does not check to make sure one Docklet is not continually unlocking however, so do not do this as it will interfere with other docklets. Only lock and unlock the mouse effect when needed.
