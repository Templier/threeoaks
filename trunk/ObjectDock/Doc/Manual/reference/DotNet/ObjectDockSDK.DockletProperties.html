Docklet Properties

The properties of the Docklet class are listed below. For a complete list of Docklet class members, see the Docklet Members topic.

Public Instance Properties

ContextMenu Gets or sets the context menu.
DockAlign Gets or sets the alignment of the dock with this docklet.
DockEdge Gets or sets the edge of the screen the dock with this docklet is located on
Image Sets the image for this docklet.
ImageFile Sets the file to use as an image for this docklet
InteropFolder Gets the relative path of the folder that contains the Interop DLL.
IsVisible Gets a value indicating whether or not this docklet is currently visible
Label Gets or sets the current label of this docklet.
Overlay Sets the image overlay for this docklet.
Rect Retrieves the bounding rectangle of this docklet in screen coordinates.
RelativeFolder Gets the relative path of the folder that contains this Docklet's DLL.
RootFolder Gets the root folder.
Window Gets the docklet Window (to pass at dialog/messagebox creation).
