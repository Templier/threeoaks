Docklet.LockMouseEffect Method 

Allows a Docklet to lock the mouse effect (e.g. zooming, etc) of its owning dock.

public void LockMouseEffect(
   bool locked
);

Parameters

locked
true to lock the mouse-over effect (zooming, etc) of the dock
false to unlock the mouse-over effect (zooming, etc) of the dock

Remarks

The mouse-over effect lock is done via a counter, so that if multiple Docklets are locking at the same time the effect will stay locked until the last Docklet unlocks. The counter does not check to make sure one Docklet is not continually unlocking however, so do not do this as it will interfere with other docklets
