DockletGetRelativeFolder

Retrieves the relative path of the folder that contains the specified Docklet's DLL.

void DockletGetRelativeFolder(
   HWND hwndDocklet,
   char *szFolder
);

Parameters

hwndDocklet
The handle to the window passed during OnCreate that owns this docklet and therefore identifies the docklet to the host.
szFolder
[out] Pointer to a string that will receive the relative location of folder that contains the specified docklet's DLL.

Remarks

The path returned includes a final backslash.

You can use this path returned to create a private INI file to store settings common to all docklets of this same type that might be added to a dock, e.g. language or regional settings, which would be constant from instance to instance of a given plugin.

In addition, you can limit the user to only picking images for this plugin from this plugin's folder by use with DockletBrowseForImage.
