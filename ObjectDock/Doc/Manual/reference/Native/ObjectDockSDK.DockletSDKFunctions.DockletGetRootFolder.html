DockletGetRootFolder

Returns the absolute location of the ObjectDock root folder, from which all things are relative.

void DockletGetRootFolder(
   HWND hwndDocklet,
   char *szFolder
);

Parameters

hwndDocklet
The handle to the window passed during OnCreate that owns this docklet and therefore identifies the docklet to the host.
szFolder
[out] Pointer to a string that will receive the absolute location of the ObjectDock root folder.

Remarks

The path returned includes a final backslash.

To turn any relative path given anywhere into an absolute path, simply copy the relative path you want to make absolute onto the end of the the path returned by this function.
