ObjectDock SDK Reference

Docklet SDK Functions

The included SDK functions allow the docklet to interact with the host, ObjectDock. To uniquely identify the plugin, you should always remember to record the "hwndDocklet" parameter given in "OnCreate", as it will be needed for the host to identify what dock item you are trying to reference when making changes or querying for information. Remember to always pass the hwndDocklet when it is asked for!

SDK Functions

DockletIsVisible Query for whether or not the specified Docklet is currently visible.
DockletGetRect Retrieves the bounding rectangle of a given Docklet in screen coordinates.
DockletGetLabel Retrieves the current label of the specified Docklet.
DockletSetLabel Changes the current label of the specified Docklet.
DockletLoadGDIPlusImageLoads an GDI+ image based on a relative path given.
DockletSetImage Allows a docklet to manually set its image to a given GDI+ Image.
DockletSetImageFile Changes the specified docklet's image to that of the given file.
DockletSetImageOverlay Sets the image overlay.
DockletBrowseForImage Shows the standard ObjectDock "Choose Image" dialog template, and allows you to select an image file.
DockletLockMouseEffect Allows a Docklet to lock the mouse effect (e.g. zooming, etc) of its owning dock.
DockletDoAttentionAnimation Causes the specified Docklet to animate in the dock to try to get the user's attention.
DockletGetRelativeFolder Retrieves the relative path of the folder that contains this Docklet's DLL.
DockletGetRootFolder Returns the absolute location of the ObjectDock root folder, from which all things are relative.
DockletDefaultConfigDialog Opens up the default configuration dialog for a given dock item, thus providing a simple way for the user to adjust a docklet's title and image.
DockletQueryDockEdge Retrieves which edge of the screen the dock with the given Docklet is located on.
DockletSetDockEdge Changes which edge of the screen the dock with the given Docklet is located on.
DockletQueryDockAlign Retrieves the alignment of the dock with the given Docklet.
DockletSetDockAlign Changes the alignment of the dock with the given Docklet.
