DockletQueryDockEdge

Retrieves which edge of the screen the dock with the given Docklet is located on.

int DockletQueryDockEdge(
   HWND hwndDocklet
);

Parameters

hwndDocklet
The handle to the window passed during OnCreate that owns this docklet and therefore identifies the docklet to the host.

Return Value

Returns which edge of the screen that the dock containing the given docklet is on.

The return values for this function are as follows:

   0 = Bottom Edge of the screen
   1 = Top Edge of the screen
   2 = Left Edge of the screen
   3 = Right Edge of the screen

Remarks

Only applicable to a normal dock (not tabbed)
