ObjectDock SDK Reference

OnDropFiles

Called by the host when a file has been dropped onto this docklet. The files being dropped are listed in the HDROP object passed.

void CALLBACK OnDropFiles(
   DOCKLET_DATA *lpData,
   HDRPP hDrop
);

Parameters

lpData
Personal data for this plugin instance returned during OnCreate. Datatype is arbitrary.
hDrop
An HDROP object that represents the file(s) dropped onto the docklet item.
The docklet should free the hDrop structure accordingly upon returning from this function.
