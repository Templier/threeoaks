OnRightButtonClick

Called by the host when the dock item that represents this docklet is right-clicked. If you choose to implement this function, you should return true to indicate you processed the message.
If you do not implement this function or return FALSE, ObjectDock will, as a default handler, display a menu allowing the user to configure the dock item.

BOOL CALLBACK OnRightButtonClick(
   DOCKLET_DATA *lpData,
   POINT *ptCursor,
   SIZE *sizeDocklet
);

Parameters

lpData
Personal data for this plugin instance returned during OnCreate. Datatype is arbitrary.
ptCursor
The position of the cursor at the time of the event, in coordinates relative to the top-left of the docklet image.
sizeDocklet
The size of the docklet image at the time of the event.

Return Value

Return TRUE to signify that this event was acted upon.
Return FALSE or do not implement to signify that this event was NOT acted upon.
In this case, for OnRightButtonClick, the default action will be to display a simple menu giving the user the opportunity to bring up an options dialog for the dock item. (This menu item will invoke OnCommand if selected by the user)
