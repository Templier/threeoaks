DockletSetDockEdge

Changes which edge of the screen the dock with the given Docklet is located on.

BOOL DockletSetDockEdge(
   HWND hwndDocklet,
   int iNewEdge
);

Parameters

hwndDocklet
The handle to the window passed during OnCreate that owns this docklet and therefore identifies the docklet to the host.
iNewEdge
Defines the new edge of the screen the dock should be docked to.

Possible values are as follows:

   0 = Bottom Edge of the screen
   1 = Top Edge of the screen
   2 = Left Edge of the screen
   3 = Right Edge of the screen

Return Value

TRUE is the function succeeded,

FALSE otherwise.

Remarks

Only applicable to a normal dock
