ObjectDock SDK Reference

OnSave

Sent to a plugin when it needs to save its individual properties. Since multiple of the same plugins can be in existence at any given time, the plugin should save options that are specific to this instance of the plugin to the INI file and section given in szIni and szIniGroup and not elsewhere.

void CALLBACK OnSave(
   DOCKLET_DATA *lpData,
   char *szIni,
   char *szIniGroup,
   BOOL bIsForExport
);

Parameters

lpData
Personal data for this plugin instance returned during OnCreate. Datatype is arbitrary.
szIni
Absolute path of an INI file which the plugin should use to save its options to for this specific plugin instance.
szIniGroup
The INI Section of an INI file which the plugin should use to save options to.
bIsForExport
Specifies whether or not this call to OnSave is the result of the docklet being packaged for distribution.
While insignificant for most plugins, if true plugins storing passwords and other such personal should not save this data.
