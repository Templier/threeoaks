DockletSetImageFile

Changes the specified docklet's image to that of the given file.

void DockletSetImageFile(
   HWND hwndDocklet,
   char *szImage
);

Parameters

hwndDocklet
The handle to the window passed during OnCreate that owns this docklet and therefore identifies the docklet to the host.
szImage
[in] Specifies the relative (or absolute) path of an image to set as the image for the specified docklet
