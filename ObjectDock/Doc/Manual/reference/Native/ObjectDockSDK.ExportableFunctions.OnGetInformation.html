ObjectDock SDK Reference

OnGetInformation

One of only a few mandatory functions, this function is called when host is looking through a folder of docklet dlls and wants to retrieve information about this specific docklet dll. Fill in all information accordingly for accurate reporting. This function, unlike all the others that you can export, is not associated with any specific instance of a plugin.

void CALLBACK OnGetInformation(
   char *szName,
   char *szAuthor,
   int *lpiVersion,
   char *szNotes
);

Parameters

szName
Buffer in which the short name of this docklet should be copied to.
szAuthor
Buffer in which the name of this docklet's author should be copied to.
lpiVersion
Pointer to an integer which represents the version number of this specific plugin.
Useful for automatic upgrades via packaging, so accurate versioning is recommended.
The version of the plugin is determined by taking the value placed into lpiVersion and dividing by 100. Thus, (*lpiVersion) = 150; says that this is v1.5 of a docklet.
szNotes
Buffer in which a more detailed description (including any sidenotes) should be copied.
