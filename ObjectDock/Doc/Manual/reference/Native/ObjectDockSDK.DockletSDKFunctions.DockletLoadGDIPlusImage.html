DockletLoadGDIPlusImage

Loads an GDI+ image based on a relative path given.

Bitmap *DockletLoadGDIPlusImage(
   char *szImage
);

Parameters

szImage
[in] The relative (or absolute) path of the image to load.
(Png, tga, and ico formats are supported explicitly. Other formats such as .psd are supported on a per-system basis.)

Return Value

Returns a pointer to the Bitmap* GDI+ image loaded for the image file passed.

Remarks

This method is much preferred to loading the image into a Bitmap* GDI+ plus object manually, for the reasons that DockletLoadGDIPlusImage can load an image based on a relative path which makes for more reliable saving, and in addition DockletLoadGDIPlusImage can load from images not supported by GDI+ such as from 32bit Icons.
