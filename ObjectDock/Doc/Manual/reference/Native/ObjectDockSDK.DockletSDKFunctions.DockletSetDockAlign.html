DockletSetDockAlign

Changes the alignment of the dock with the given Docklet.

BOOL DockletSetDockAlign(
   HWND hwndDocklet,
   int iNewAlign
);

Parameters

hwndDocklet
The handle to the window passed during OnCreate that owns this docklet and therefore identifies the docklet to the host.
iNewAlign
Defines the new alignment of the dock with the specified docklet.

Possible values are as follows:

   0 = Align left (or top, if dock is vertical)
   1 = Align middle
   2 = Align right (or bottom, if dock is vertical)

Return Value

TRUE is the function succeeded,

FALSE otherwise.

Remarks

Only applicable to a normal dock (not tabbed)
