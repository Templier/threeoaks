DockletIsVisible

Query for whether or not the specified Docklet is currently visible.

BOOL DockletIsVisible(
   HWND hwndDocklet
);

Parameters

hwndDocklet
The handle to the window passed during OnCreate that owns this docklet and therefore identifies the docklet to the host.

Return Value

TRUE if the Docklet specified is currently visible.

FALSE if the Docklet specified is currently hidden
