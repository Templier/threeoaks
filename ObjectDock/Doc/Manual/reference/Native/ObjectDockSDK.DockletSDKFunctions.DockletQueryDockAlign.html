DockletQueryDockAlign

Retrieves the alignment of the dock with the given Docklet.

int DockletQueryDockAlign(
   HWND hwndDocklet
);

Parameters

hwndDocklet
The handle to the window passed during OnCreate that owns this docklet and therefore identifies the docklet to the host.

Return Value

Returns the alignment of the dock containing the given docklet.

The return values for this function are as follows:

   0 = Align left (or top, if dock is vertical)
   1 = Align middle
   2 = Align right (or bottom, if dock is vertical)

Remarks

Only applicable to a normal dock (not tabbed)
