ObjectDock SDK Reference

DockletSetImage

Allows a docklet to manually set its image to a given GDI+ Image.

void DockletSetImage(
   HWND hwndDocklet,
   Image *lpImageNew,
   BOOL bAutomaticallyDeleteImage = TRUE
);

Parameters

hwndDocklet
The handle to the window passed during OnCreate that owns this docklet and therefore identifies the docklet to the host.
lpImageNew
[in] Specifies the new base image for the docklet.
bAutomaticallyDeleteImage
Upon the next call to DockletSetImage, DockletSetImageFile, etc, ObjectDock would by default delete the image set to it by this function. Set this flag to FALSE to prevent ObjectDock from deleting this image, thus allowing this image to be reused after it is no longer being displayed. By default, set to TRUE - yes delete the image after no longer in use.
