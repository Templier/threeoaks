IDockletInterface.OnCreate Method 

This function is called every time a docklet from this dll is created. Provides the handle of the window the docklet will be represented by, along with the instance of the docklet's dll. If the plugin is being loaded for the first time, szIni and szIniGroup will be NULL, however otherwise they will specify the ini file and ini section the docklet should load its individual options from.

void OnCreate(
   DOCKLET_STATIC_DATA data,
   string ini,
   string iniGroup
);

Parameters

data
The data structure containing info about the docklet
ini
Absolute path of an INI file which the plugin should use to load options from.
Can be 0. If 0, then the plugin is a new plugin and default options should be loaded
iniGroup
The INI Section of an INI file which the plugin should use to load options from.
Can be 0. If 0, then the plugin is a new plugin and default options should be loaded.
