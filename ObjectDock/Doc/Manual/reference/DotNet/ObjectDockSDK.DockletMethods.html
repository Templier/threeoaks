Docklet Methods

The methods of the Docklet class are listed below. For a complete list of Docklet class members, see the Docklet Members topic.

Public Instance Methods

BrowseForImage Shows the standard ObjectDock "Choose Image" dialog template, and allows you to select an image file.
DefaultConfigDialog Opens up the default configuration dialog for a given dock item, thus providing a simple way for the user to adjust a docklet's title and image.
DoAttentionAnimation Causes the specified Docklet to animate in the dock to try to get the user's attention.
GetCoordinates Get the coordinates where the file(s) were dropped.
GetFiles Get the names of the dropped file(s).
LockMouseEffect Allows a Docklet to lock the mouse effect (e.g. zooming, etc) of its owning dock.
ShowContextMenu Shows the context menu associated to this docklet
