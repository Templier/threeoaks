IDockletInterface.OnConfigure Method 

Called by the host in the event OnRightButtonClick returned FALSE or was not implemented, and it was selected to configure the dock item. If you do not implement this function or return false, ObjectDock will, as a default handler, will open a basic configuration dialog allowing the dock item's text and image to be changed. To override this functionality, implement it and return true.

bool OnConfigure();

Return Value

Return true to signify that this event was acted upon.
Return false or do not implement to signify that this event was NOT acted upon.
In this case, for OnConfigure, a default configuration dialog will be shown, which will allow the user to change the docklet's caption and image.
