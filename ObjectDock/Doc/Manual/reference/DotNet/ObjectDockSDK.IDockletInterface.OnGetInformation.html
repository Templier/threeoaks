IDockletInterface.OnGetInformation Method 

This function is called when host is looking through a folder of Docklet DLLs and wants to retrieve information about this specific Docklet DLL.

void OnGetInformation(
   out string name,
   out string author,
   out int version,
   out string notes
);

Parameters

name
Name of the Docklet
author
Author of the Docklet
version
Version of the Docklet
The version of the Docklet is determined by taking the value placed into version and dividing by 100. Thus, version = 150; says that this is v1.5 of a Docklet
notes
More detailed description (including any side notes) of the Docklet
