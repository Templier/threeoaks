IDockletInterface.OnExportFiles Method 

Sent to the docklet when the docklet is to be packaged up as part of a theme to redistribute.

string[] OnExportFiles();

Return Value

The relative paths of the files which this docklet depends upon, and which should be included if the plugin is packaged up for redistribution.

Remarks

Paths are relative to the .Net Docklet directory
