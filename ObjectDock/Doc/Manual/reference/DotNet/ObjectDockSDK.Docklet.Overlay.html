Docklet.Overlay Property

Sets the image overlay for this docklet.

public System.Drawing.Image Overlay {set;}

Property Value

Remarks

In ObjectDock, instead of dealing with Paint messages, ObjectDock allows you to set 'overlay' image over the top of the docklet base image. This allows for such things as a base background image to be constantly loaded, and then for details to be added on top of this image, for example a CPU meter on top of a holder, or a mail notification icon with a number on top of a mail icon loaded from a file, which will not have to be reloaded when mail arrives and the overlay image needs to be changed.
